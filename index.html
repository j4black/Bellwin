我們致力於提供高品質的電源管理產品與OEM/ODM服務。
我們一直在尋求更好的電源控制和管理方式，以確保客戶的使用安全和滿意度。

ZD系列

ZD系列產品內建新研發功率晶片，可取得電壓、電流、功率，瓦時及功因等資料；LCD數位電表可查看耗電量如電壓、電流及IP等資料，可偵測及顯示各路電流，並可由LCD面板操作各路電源開關。可附加數位溫濕度顯示器，依環境溫濕度變化控制電源開關。支援SNMP(V1/V2/V3)協議、SNMP Trap，另提供 API 方便使用者開發應用程式。

SmartMonitor系列

SmartMonitor 系列SmartMonitor 系列產品不僅可與Hotswap PDU系列並且與Metered PDU系列結合，達到PDU串接功能，讓使用者針對多設備進行遠端監控。；SmarMonitor 系列產品可與中控軟件-IDC View搭配使用，透過中控軟件，管理者可統計連接設備每月的電量使用情況。

SP系列 ( 接單生產 )

SP 系列產品具備多種電力監控功能，包括電源控制、電力監測、溫度監測、UPS電力監測、遠程電源網絡管理、語音控制、短訊與Email告警服務、和支持API控制。

SG系列 ( 接單生產 )

SG 系列產品不僅支持SG 系列的產品功能，並內建特殊電力芯片且支持KW/H電力計量以提供更多電力監測數據供使用者參考；SG 系列產品可與中控軟件-IDC View搭配使用，透過中控軟件，管理者可統計連接設備每月的電量使用情況。

EnvirPro系列

EnvirPro 系列產品支援擴充溫濕度感測器,可偵測7組温度及一組濕度，即時監測外在環境溫濕度。提供乾接點輸入與乾接點輸出；監控周圍溫濕度、漏水、煙霧等會傷害使用者設備；亦達到即時告警，確保設備安全。支援12伏直流電乾接點輸出開啟或關閉設備；偵測器不須另外接電。可偵測UPS低電量，讓設備能及時關機，防止斷電損壞設備。

IP Metered PDU

IP Metered PDU 系列產品可直接與您的電腦做網路連線，不需另外連接網路線。具電力監控功能，LCD 面版可查看電流，電壓與瓦時功能，功能鍵可設定告警電流。提供在不斷電的情形下更換顯示模組，具有電流過載保護器，防止電流過載損壞設備。支援 SNMP V1, V2, V3，可由 SNMP View 程式監看機櫃監測主機狀態。

HotSwap PDU

HotSwap 系列產品具電力監控功能，LCD 面版可查看 PDU電壓、總電流、頻率及分支電流等資訊，功能鍵可設定總電流告警電流及分支告警電流。提供在不斷電的情形下更換顯示模組，具有電流過載保護器，防止電流過載損壞設備。

Metered PDU

Metered PDU 系列產品具電力監控功能，LED 面版可查看 PDU總電流，功能鍵可設定總電流告警電流；可察看上次最高之告警電流值。具有電流過載保護器，防止電流過載損壞設備。RJ11插頭可將電流資料傳送至主機中。

Basic PDU

Basic PDU 系列產品具有電流過載保護器，防止電流過載損壞設備。

LCD顯示螢幕之溫溼度感測器。適用產品: ZD系列，NSH，EHP-2

UP系列

由電腦 USB 控制的電源控制器，一台電腦可同時串接多台電源控制器。另提供 API 方便使用者開發應用程式。
